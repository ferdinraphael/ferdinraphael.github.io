# SEO Checklist

For every page:

- [ ] Unique title
- [ ] Unique description
- [ ] One clear H1
- [ ] Logical H2/H3 sections
- [ ] Clean permalink
- [ ] Internal links where useful
- [ ] Contact CTA where useful
- [ ] No keyword stuffing
- [ ] Useful content first

For every blog post:

- [ ] Practical question or problem
- [ ] Clear category
- [ ] Useful description
- [ ] Related service/resource CTA if relevant
- [ ] Response CTA instead of comments
