# Setup Guide

## 1. Copy the starter

Copy this starter into a new Jekyll site repository.

## 2. Update `_config.yml`

Change:

- title
- description
- author
- navigation
- analytics settings if needed

## 3. Update pages

Replace demo pages:

- `index.md`
- `blog.md`
- `contact.md`

## 4. Add posts

Create posts inside:

```text
_posts/
```

Use the naming format:

```text
YYYY-MM-DD-post-title.md
```

## 5. Run locally

```bash
bundle install
bundle exec jekyll serve
```

## 6. Publish

Push to GitHub and enable GitHub Pages where needed.
