## Local Setup

Install dependencies:

```bash
bundle install
```

Run locally:

```bash
bundle exec jekyll serve
```

Open:

```bash
http://localhost:4000
```

## Notes

This site uses Jekyll and GitHub Pages.

No backend, payment system, comments, or complex analytics are included in Phase 1.
