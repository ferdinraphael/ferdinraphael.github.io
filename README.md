# Ferdin Raphael

Personal website and writing hub for Ferdin Raphael.

Built with Jekyll and hosted on GitHub Pages.

## Live site

<https://ferdinraphael.github.io/>

## Current status

This site is in its early foundation stage.

It currently includes:

- Home page
- Writing page
- Contact page
- Writing sequence
- Basic SEO metadata
- Dark mode support
- GoatCounter + GA4 analytics

## Local Development

Install dependencies:

```bash
bundle install
```

Run locally:

```bash
bundle exec jekyll serve
```

Build the site:

```bash
bundle exec jekyll build
```

Open:

```bash
http://localhost:4000
```

## Notes

This site should remain a quiet personal hub. This site will grow gradually through writing, small experiments, and practical work.
